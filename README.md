# Day 66 — Weather Dashboard (Live API)

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A live weather dashboard using OpenWeatherMap API
with 5-day forecast, hourly data and weather maps.

## How to view
Open `index.html` in your browser.
Get a free API key from openweathermap.org

## Features
- Live weather by city or GPS location
- 5-day forecast with hourly breakdown
- Wind, humidity, visibility, UV index
- Weather condition animations
- Search history
